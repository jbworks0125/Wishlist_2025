# Christmas Wishlist Project

## Overview
This project is a responsive website for a Christmas wishlist. It allows users to view and manage their wishlist items in a festive and user-friendly interface.

## Project Structure
```
christmas-wishlist
├── src
│   ├── index.html
│   ├── css
│   │   └── styles.css
│   ├── js
│   │   └── script.js
│   └── assets
│       └── images
├── package.json
└── README.md
```

## Getting Started

### Prerequisites
- Node.js and npm installed on your machine.

### Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd christmas-wishlist
   ```
3. Install the dependencies:
   ```
   npm install
   ```

### Running the Project
To start the project, open `src/index.html` in your web browser. You can also use a local server for better performance.

### Features
- Add items to your wishlist.
- Remove items from your wishlist.
- Responsive design for mobile and desktop views.
